###### ADP1

## Installation
Install PyCoinGeckoAPI:
**pip install pycoingecko**
and clone
```
git clone https://github.com/moralfager/ADP1.git
cd adp1.git
```
## Usage
```python
from pycoingecko import CoinGeckoAPI
cg = CoinGeckoAPI()
```
